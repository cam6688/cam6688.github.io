Pacman

PythonAIReinforcement LearningInference

In my Introduction to Artifical Intelligence class, we did 4 projects that built upon the topics of AI by implementing them into a Pacman game. We first utilized search algorithms to teach Pacman how to eat all the pellets on a board, then used reinforcment learning and reward assignments to prevent Pacman from being eaten by ghosts. We eventually made our Pacman predict ghost locations, and finally had Pacman successfully win all games with scores over 1000, in a full implementation of the game's rules. You can learn more about the projects that I did for this class from the course website here.

Handwriting Classifier

PythonMachine Learning

I built an artificial neural network to classify numbers by training off of the MNIST handwritten digit database. By implementing a linear regression model to train the data, I gradually added more layers to create a complex network that could have a 97% or greater test accuracy. Finally, I redesigned the network by adding extra layers and training to classify the language that words were written in (English, Spanish, Finnish, Dutch, and Polish), which had over 89% test accuracy.
